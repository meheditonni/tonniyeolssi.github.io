Introducing Pico.

A stupidly simple & blazing fast, flat file CMS.

Making the web easy.

Stupidly Simple

Pico makes creating and maintaining and website as simple as editing text files.

Blazing Fast

Pico is seriously lightweight and doesn't use a database,making it super fast.

No Database

Pico is a "flat file" CMS,meaning no database woe's,no MySQL queries,nothing.

Markdown Formatting

Edit your website in your favourite text editor using simple Markdown formatting.

Twig Tamplates

Pico uses the Twig tamplating engine for powerful and flexible themes.

Open Sources

Pico is completely free and open source,relased under the MIT license.

Want to Contribute?

Help make Pico better by checking out the GitHub repository and submitting pull requests. If you find a bug please report it on the issues page.
